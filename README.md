# 我的待辦清單 PWA

這是用安裝精靈產生的個人待辦清單網頁程式。

## 檔案說明
- index.html：主程式（已含你的 GAS 網址）
- sw.js：離線快取
- manifest.json：PWA 設定
- icon.png：App 圖示

## 上傳步驟
1. 把這個資料夾內所有檔案上傳到你的 GitHub repo
2. 在 repo 的 Settings → Pages 啟用 GitHub Pages
3. 等 1-2 分鐘、用手機 Safari 開啟你的 Pages 網址
4. 點分享 → 加入主畫面
